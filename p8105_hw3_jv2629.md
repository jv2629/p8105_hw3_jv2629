**Make a “spaghetti plot” that shows the number of locations in each state from 2002 to 2010.**

``` r
spag_plot = beh_risk %>% 
  group_by(year, state) %>% 
  distinct(county, .keep_all = TRUE) %>%  
  summarise(county = n()) 
ggplot(data = spag_plot, aes(x = year , y = county, group = state, color = state)) + geom_line()
```
